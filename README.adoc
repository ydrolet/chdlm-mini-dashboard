== Use OAuth refresh token to have a service account
https://justin.poehnelt.com/posts/google-user-credentials-as-application-default/
```
gcloud auth application-default login --client-id-file="client_id_file.json" --scopes="https://www.googleapis.com/auth/spreadsheets,https://www.googleapis.com/auth/gmail.send"
```
